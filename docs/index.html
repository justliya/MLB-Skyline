Skyline - Revolutionizing Baseball Fan Engagement

🚀 Introduction

Skyline is an AI-powered mobile app that enhances the baseball experience by providing insights, predictive analytics, and strategic breakdowns of plays.

💡 Inspiration

We wanted to help fans understand and enjoy baseball beyond just watching. Inspired by discussions on Reddit where fans struggled with complex game decisions, Skyline turns each game into an interactive and educational experience.

📌 Features

  • Real-Time Strategy Insights: Understand game-changing plays and strategies as they happen.
  • Key Analytics: Get pre-game insights, batter-pitcher matchups, and win probability predictions.
  • Casual Mode: Simple game breakdowns for new fans.
  • Predictive Features: AI-driven forecasts for win probabilities, home runs, and impactful plays.

⚙️ Technology Stack

  • Google Cloud - BigQuery, Vertex AI, Cloud Run
  • React Native for mobile app development
  • Firebase Authentication & Firestore for user data management
  • Google Imagen for generating custom UI assets

🛠️ How We Built It

We processed historical MLB data using BigQuery and trained AI models in Vertex AI for win probability and play analysis. The app seamlessly fetches live game data from the MLB Stats API and presents it to users with AI-generated insights.
